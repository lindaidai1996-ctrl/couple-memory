# 字体文件说明

本目录存放自托管的中文字体文件，需要手动下载放入：

## 需要的文件

1. **AlibabaPuHuiTi-Regular.woff2** — 阿里巴巴普惠体 Regular
2. **AlibabaPuHuiTi-Medium.woff2** — 阿里巴巴普惠体 Medium
3. **AlibabaPuHuiTi-Bold.woff2** — 阿里巴巴普惠体 Bold
4. **SmileySans-Oblique.woff2** — 得意黑

## 下载地址

- 阿里巴巴普惠体：https://www.alibabafonts.com/#/font
- 得意黑：https://github.com/atelier-anchor/smiley-sans/releases

## 注意

- 下载后需转换为 woff2 格式（如果原始格式不是）
- 中文字体较大，建议使用 font-spider 或 cn-font-split 做子集化
- 这些字体均为免费商用许可
