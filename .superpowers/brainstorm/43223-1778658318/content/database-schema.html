Entity Relationship Diagram
User
PK id String (cuid)
email String @unique
password String (hashed)
name String
avatar String?
createdAt DateTime
→ couples CoupleUser[]
Couple (Tenant)
PK id String (cuid)
slug String @unique
name String ("我们的故事")
startDate DateTime?
coverImage String?
theme String @default("minimal")
isPublic Boolean @default(true)
plan Enum (FREE/PRO)
createdAt DateTime
→ members CoupleUser[]
→ albums Album[]
→ photos Photo[]
→ milestones Milestone[]
CoupleUser (关联表)
PK id String
FK userId → User
FK coupleId → Couple
role Enum (OWNER/PARTNER)
nickname String?
joinedAt DateTime
Album
PK id String
FK coupleId → Couple
title String
description String?
coverPhotoId String?
sortOrder Int
createdAt DateTime
→ photos Photo[]
Photo (核心表)
PK id String
FK coupleId → Couple
FK albumId → Album?
FK uploadedBy → User
— 图片信息 —
originalUrl String (OSS)
thumbnailUrl String
mediumUrl String
width Int

— EXIF 数据 —

latitude Float?
longitude Float?
camera String?
lens String?
exifData Json? (完整EXIF)
— AI 生成 —
aiCaption String?
userCaption String?
aiLayout String?
sortOrder Int
createdAt DateTime
Milestone (里程碑)
PK id String
FK coupleId → Couple
title String ("第一次旅行")
description String?
date DateTime
icon String?
location String?
coverPhotoId String?
sortOrder Int
关系说明
User ←M:N→ Couple (通过 CoupleUser)
Couple ←1:N→ Album
Couple ←1:N→ Photo

Couple ←1:N→ Milestone
所有表通过 coupleId 实现租户隔离
